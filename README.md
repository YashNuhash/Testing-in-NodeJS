## Installation ![](https://i.imgur.com/waxVImv.png)

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
# Run Locally:

Clone the project

```bash
  git clone https://github.com/YashNuhash/Testing-in-NodeJS
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
